# Lista de Tarefas

Aplicação web para gerenciar tarefas com persistência local.

## Funcionalidades

- Adicionar tarefas
- Remover tarefas
- Persistência usando localStorage
- Contador de tarefas

## Tecnologias

- HTML
- CSS
- JavaScript Vanilla

## Como usar

Abra o arquivo `index.html` no navegador.
